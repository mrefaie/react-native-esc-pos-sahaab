# react-native-esc-pos-sahaab

ESC-POS Printer Module

## Installation

```sh
npm install react-native-esc-pos-sahaab
```

## Usage

```js
import EscPosSahaab from "react-native-esc-pos-sahaab";

// ...

const result = await EscPosSahaab.multiply(3, 7);
```
